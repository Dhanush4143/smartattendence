# Smart Attendance - Anti-Gravity Research Hub

A comprehensive attendance management system built with **Java Spring Boot** backend and **React/Vite** frontend.

## 🔗 Project Links

- **GitHub Repository**: https://github.com/Dhanush4143/smartattendence
- **Frontend Deployment**: Coming soon...

## 📋 Project Overview

Smart Attendance is a modern attendance tracking system designed for research and educational institutions. It features:

- ✅ User authentication with JWT
- ✅ Attendance tracking and reporting
- ✅ Module/Course management
- ✅ Bulk attendance upload
- ✅ Role-based access control
- ✅ MongoDB database integration

## 🚀 Quick Start

### Backend Setup
```bash
cd antigravity-backend
mvn clean install
mvn spring-boot:run
```

### Frontend Setup
```bash
cd antigravity-frontend
npm install
npm run dev
```
